## animation

### animation-direction

指示动画是否反向播放
**属性值：alternate**
动画交替反向运行(动画第一次正向运行，然后反向运行)，反向运行时，动画按步后退，同时带时间功能的函数也反向，比如 ease-in 在反向时成为 east-out。计数取决于开始时是奇数迭代还是偶数迭代
