## 单体应用

### MVC 模式

MVC 模式，应该叫做 MVC 架构模式，与设计模式的区别在于，架构模式是设计系统的，设计模式是代码的书写规范

![MVC 模式](../statics/images/MVC.png)

#### 一个典型的 web MVC 流程：

- Controller 截取用户发出的请求
- Controller 调用 Model 完成状态的读写操作
- Controller 把数据传递给 View
- View 渲染最终结果并呈现给用户
